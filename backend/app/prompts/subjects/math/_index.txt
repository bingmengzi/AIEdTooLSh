## 数学学科专项指导

你正在处理一道数学题目或概念。数学可视化的核心是将抽象概念转化为直观图形。

### 数学可视化原则

1. **图形精确性**
   - 几何图形必须符合数学定义（如直角必须是90度）
   - 数值计算必须准确（使用正确的数学公式）
   - 坐标系统清晰可辨（原点、轴向、刻度）

2. **概念可视化策略**
   - 几何证明：展示关键辅助线、标注角度/边长
   - 函数图像：支持缩放、平移、参数调节
   - 数列/级数：动态展示求和过程
   - 计算推导：步骤卡片式展开

3. **数学公式渲染**
   - 简单公式：使用 Canvas 自绘或 HTML 上标下标
   - 复杂公式：引入 MathJax 3 CDN 渲染
   - 公式要与图形元素保持视觉关联

### 数学交互设计要点

1. **参数可调**：让用户调整边长、角度、系数等参数，观察变化规律
2. **过程可视**：证明/计算过程可逐步展开，配合图形联动
3. **结论突出**：关键等式、定理结论用高亮框强调

### Canvas绑定数学坐标系技巧

```javascript
// 建立数学坐标系（原点在画布中央，Y轴向上）
const originX = canvas.width / 2;
const originY = canvas.height / 2;
const scale = 40; // 每单位像素数

function toCanvasX(mathX) { return originX + mathX * scale; }
function toCanvasY(mathY) { return originY - mathY * scale; } // 注意Y轴翻转
```
