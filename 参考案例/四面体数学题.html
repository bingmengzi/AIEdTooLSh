空间向量的应用：四面体对棱垂直定理

尝试拖动左侧图形，多角度观察

三维模型

单指旋转 / 双指缩放

证明题

在四面体 ABCD 中，已知：
AB ⊥ CD
AD ⊥ BC
证明：AC ⊥ BD

推理过程 (向量法)

1

建立基底向量

设点 A 为空间原点，定义三个基底向量：
b = AB,    c = AC,    d = AD

则其他棱的向量可表示为：
CD = d - c
BC = c - b
BD = d - b

2

转化已知条件（垂直 ⇔ 内积为0）

由已知条件展开内积：
① 因为 AB ⊥ CD，所以：
b · (d - c) = 0   ⇒   b · d = b · c

② 因为 AD ⊥ BC，所以：
d · (c - b) = 0   ⇒   d · c = b · d

3

推导结论边对应的内积

综合①②可得核心等式：
b · c = b · d = c · d
现在考察目标边 AC 和 BD 的内积：
AC · BD = c · (d - b)
= c · d - c · b

4

得出结论

因为前面已证 c · d = b · c (即 c · b)，
所以代入上式得：
AC · BD = c · d - c · b = 0

因此 AC ⊥ BD，证明完毕！
