推导与课堂思考题

✅ 答案：逐渐变大。

向左走 → 人的逆时针力矩增大 → CD杆必须提供更大的向上托力 → AB对CD向下压力增大 → 压敏电阻受压增大 → 阻值减小 → 总电阻减小 → 电流变大。

✅ 答案：减小了 5Ω。

根据力矩平衡，向左走 1m 导致 C 点需增加向上托力 300N × 1m / 3m = 100N，从而压敏电阻受到的压力增加 100N。

结合右侧 R-F 图象（斜率 -0.05），压力增加 100N，阻值对应减少 5Ω。

1. 提取已知条件：定值电阻 R₀=10Ω。
由第(2)问可知，向左走1m后，电阻减小了 5Ω。

2. 两次状态列方程：
初始状态电流 0.2A： U / 0.2A = R₁ + 10
移动后电流 0.24A： U / 0.24A = (R₁ - 5) + 10

3. 联立求解：
化简两式：5U = R₁ + 10； (25/6)U = R₁ + 5
两式相减得：(5 - 25/6)U = 5 ➔ 解得 U = 6V。
(同时代回可得初始点 R₁ = 20Ω)

✅ 答案：1.5 W。

(注：有了 U=6V，请拖动上方小人验证！)

在A点(最右端)：人在支点右侧1m。产生顺时针力矩，C点(位于O左侧)被迫向下拉100N。导致向下总压 F = 200 - 100 = 100N。
查函数得出 R = 35 - 0.05(100) = 30Ω。电路总功率 PA = U² / R总 = 36 / 40 = 0.9 W。

在B点(最左端)：人在支点左侧4m。C点提供极大的向上托力400N。导致向下总压 F = 200 + 400 = 600N。
查函数得出 R = 35 - 0.05(600) = 5Ω。此时总功率 PB = U² / R总 = 36 / 15 = 2.4 W。

变化量 ΔP = 2.4W - 0.9W = 1.5 W。
